# CLAUDE.md — Wayback-Diff

## Overview
CLI tool for intelligent web page comparison with Wayback Machine support. Automatically strips archive injection artifacts, scores change significance, and generates visual regression diffs across multiple browsers.

## Tech Stack
- Python (CLI via Click)
- Selenium / multi-browser (Chrome, Firefox, Edge, Opera)
- PyPI package (`wayback-diff`)
- Docker
- pytest (testing)
- Codecov (coverage)

## Development
```bash
# Install for development
pip install -e ".[dev]"

# Run tests
pytest

# Run the CLI
wayback-diff https://old.example.com https://new.example.com --visual --markdown
```

## Architecture
- `wayback_diff/` — Main package
  - `cli.py` — Click CLI entry point
  - `diff_engine.py` — Core comparison logic
  - `fetcher.py` — Page fetching
  - `wayback_cleaner.py` — Strips Wayback Machine injection artifacts
  - `visual_comparison.py` — Multi-browser screenshot diffing
  - `report_generator.py` — Markdown report output
  - `link_traverser.py` — Link following for multi-page comparison
- `tests/` — Test suite
- `docs/` — Documentation

## Key Rules
- CI/CD exit codes: 0 = no changes, 1 = low/medium, 2 = high significance
- Always clean Wayback Machine artifacts (banners, analytics, URL rewrites) before comparing
- Published to PyPI as `wayback-diff`
- Docker image available for containerized usage
